Quick Start Documentation 🚀

  1. Fork the GitHub Repository.
  2. Create a new file inside the domains/ folder named exactly after your desired subdomain (e.g., yourname.json).
  3. Paste the following JSON structural schema template into your file: 
    {
  "owner": {
    "username": "your-github-handle",
    "email": "hello@example.com"
  },
  "records": {
    "CNAME": "your-username.github.io"
  }
}
  4. Submit a Pull Request (PR). Once our automated pipeline passes validation and an admin merges it, your domain will be live within 5 minutes!
